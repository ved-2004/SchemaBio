Burkholderia flipping program — internal project notes

Status: Data collated; mechanism characterization phase.

Findings so far
---------------
- L1 and L2 show high-level FQ resistance (cipro/levo) in some isolates; L3 mixed.
- ATCC 25609 used as reference (WT). Fold-shifts up to 64× observed for cipro in L1/L2.
- Compound screen: Bcep-Inh-01 (28 nM), Bcep-Inh-05 (45 nM) look promising vs gyrA; need testing vs flipping isolates.
- VCF: gyrA D87N-like, parC S80/E84, acrB variant called — consistent with FQ/efflux story.

Open questions
--------------
- Is flipping reproducible across batches? Only 2 replicates in current MIC.
- Missing: vehicle/DMSO control in compound screen for some batches.
- No formal time-kill or synergy (checkerboard) yet — recommended next analyses.
- Need n≥3 replicates and expanded isolate coverage for publication.

Missing controls / gaps
------------------------
- No vehicle control well explicitly flagged in compound screen.
- Single or dual replicate in part of assay set (n=2 only).
- No target engagement or enzyme inhibition (WT vs mutant) data yet.
- No ADMET or manufacturability data in this stack.

Recommended next steps
----------------------
1. Add time-kill for Bcep-Inh-01 vs L1_003, L2_003.
2. Run checkerboard (FQ + Bcep-Inh-01) to probe synergy.
3. Expand to ≥5 isolates per lineage; add meropenem/ceftazidime where relevant.
4. Document CRO options for external validation and translational planning.

Reproducibility
---------------
- Assay type: MIC broth microdilution (CLSI M07).
- Replicate count: 2 for MIC; compound screen batches B1, B2.
- Concern: expanded isolate testing needed before claiming flipping prevalence.
